# Docuchat-Chatbot-main
Docuchat_Chatbot_Gemini
DocuChat is an AI-powered document chatbot designed to help users interact intelligently with their documents. The application allows users to upload PDF and DOCX files and ask questions directly based on the content of those documents. By leveraging Google Gemini (Generative AI), DocuChat understands the context of the uploaded files and provides accurate, human-like responses through an intuitive Streamlit-based web interface.
